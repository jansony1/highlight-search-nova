# 用户使用指南

## 功能概述
Amazon Nova Multimodal Embedding测试平台支持上传和搜索多种类型的媒体文件，包括图像、视频、音频和文本。

## 快速开始

### 1. 访问网站
在浏览器中打开 `http://localhost:5000`

### 2. 上传文件

#### 图像上传
1. 点击"📷 图像"标签
2. 点击上传区域或拖拽图像文件
3. 可选：添加描述文本
4. 点击"上传并生成Embedding"

#### 视频上传
1. 点击"🎥 视频"标签
2. 选择视频文件（支持MP4, AVI, MOV, WMV）
3. 可选：添加描述文本
4. 点击上传按钮

#### 音频上传
1. 点击"🎵 音频"标签
2. 选择音频文件（支持MP3, WAV, FLAC, M4A）
3. 可选：添加描述文本
4. 点击上传按钮

#### 文本输入
1. 点击"📝 文本"标签
2. 在文本框中输入内容
3. 点击上传按钮

### 3. 搜索内容
1. 在搜索框中输入查询文本
2. 点击"🔍 搜索相似内容"
3. 查看搜索结果和相似度分数

## 功能特性

### 多模态支持
- **图像**: 支持常见图像格式，可结合文本描述
- **视频**: 自动提取视频特征，支持文本标注
- **音频**: 处理音频内容，生成语义向量
- **文本**: 直接文本输入和处理

### 智能搜索
- **语义搜索**: 基于内容语义而非关键词匹配
- **跨模态搜索**: 用文本搜索图像、视频等
- **相似度评分**: 显示搜索结果的相关性分数

### 用户界面
- **Amazon风格设计**: 专业的用户体验
- **响应式布局**: 支持桌面和移动设备
- **实时反馈**: 上传和搜索进度提示

## 使用技巧

### 1. 优化搜索结果
- 使用具体的描述性词汇
- 尝试不同的查询表达方式
- 结合上下文信息

### 2. 文件上传建议
- 为媒体文件添加描述文本可提高搜索准确性
- 确保文件质量良好
- 避免上传过大的文件

### 3. 最佳实践
- 定期清理不需要的文件
- 使用有意义的文件名
- 保持描述文本简洁明确

## 限制说明

### 文件限制
- 最大文件大小: 100MB
- 支持的文件格式有限
- 处理时间取决于文件大小

### 功能限制
- 搜索结果最多显示5个
- 不支持批量操作
- 需要稳定的网络连接

## 故障排除

### 上传失败
- 检查文件格式是否支持
- 确认文件大小未超限
- 重试上传操作

### 搜索无结果
- 尝试不同的查询词
- 确保已上传相关内容
- 检查网络连接

### 页面加载问题
- 刷新浏览器页面
- 清除浏览器缓存
- 检查服务器状态

## 技术支持
如遇到问题，请检查：
1. 浏览器控制台错误信息
2. 网络连接状态
3. 服务器运行状态
